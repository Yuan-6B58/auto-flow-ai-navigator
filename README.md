 | 分类 | 工具数 | 代表工具 |
|------|-------|---------|

| AI Agent 框架 | 5 | AutoGPT, CrewAI, LangGraph |
| AI 模型工作流 | 4 | ComfyUI, LangFlow, PromptFlow |

 直接用浏览器打开 `index.html` 即可。
